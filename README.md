# Rysunek-2D
Rysunek 2D z użyciem funkcji graficznych biblioteki turtle w Pythonie. Opiera się o pętle rysującą koła w lewo od miejsca startu żółwia i kończy się w momencie, kiedy powstaje spirograf.
